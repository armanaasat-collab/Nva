Құпия код (PIN)
Қолданбаны ашу үшін 1-8 сандық код қоюға болады. Код қойылған соң, қолданбаны әр ашқанда сол код сұралады.
API кілттер
Бірнеше кілт қосуға болады (мысалы, 10-20 кілт). Біреуінің лимиті бітсе немесе сервер жүктелсе, қолданба автоматты түрде келесі кілтке ауысады — сұхбатыңыз үзілмейді, тек фондағы кілт ауысады. Әр кілтті жаңа жолға жазыңыз.
